repositionItems()method 
public function repositionItems():void

Repositions arranger items if any exists.

If you are using dynamicItemsDimensions = false (by default) then you should use maxWidth and maxHeight parameters and set the needed grid size for the control of grid elements arrangement.

If you are using dynamicItemsDimensions = true then you should use rows and columns parameters and set the needed number for the control of grid elements arrangement.

selectItemRendererByIndex()method 
public function selectItemRendererByIndex(itemIndex:int, selectAndClick:Boolean = true):IItemRenderer

Select IItemRenderer by index. If there is an instance of IItemRenderer with the specified index in itemsCollection then "selected = true" will be bound and selection of the previous element will be reset. If itemIndex is equal -1 and any IItemRenderer from the itemsCollection list is selected, selection will be reset. If IItem Renderer with the specified index won't be found, null will be returned.

Parameters
itemIndex:int — int index
 
selectAndClick:Boolean (default = true) — Boolean States if the ItemRendererEvent.ITEM_CLICK event should be dispatched after selection or not.

Returns
IItemRenderer — IItemRenderer Selected element; element whose selection was reset if itemIndex == -1
