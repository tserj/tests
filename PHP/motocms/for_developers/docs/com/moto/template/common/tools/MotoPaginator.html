Removes arranger from the paginator arrangers collection and disconnets it from the paginator events. It doesn't change arranger itemsToShow property to its previous value, you should set it manually.

public function showPageNumbers(target:Sprite, itemRenderer:String, numbersPerPage:uint, areaRectangle:Rectangle = null):void

Shows page numbers. Creates arranger and paginator for them.

Parameters
target:Sprite — Holder sprite for page numbers item renderers
 
itemRenderer:String — Item renderer class name
 
numbersPerPage:uint — How many numbers to show per page
 
areaRectangle:Rectangle (default = null) — Area to show page numbers in. If it is not null page numbers are automatically aligned to center.
