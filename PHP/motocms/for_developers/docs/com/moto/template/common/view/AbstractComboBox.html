labelFieldproperty 
public var labelField:String

Field name in the dataProvider array elements for display as a label in the TextInput and the roll-out list. By default, the control element uses the toString() method for every Array object and displays it. But if the dataProvider elements do not return toString(), the labelField property can be set for using another property.
