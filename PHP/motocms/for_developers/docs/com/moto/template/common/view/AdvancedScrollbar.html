AdvancedScrollbar Class - adds scrolling logic for the sprite using different scrolling methods.

The main features of the scrollbar class are:

  • scrolling logic can be easily added without any scrollbar specific controls and doesn't require any other objects than content holder mask rectangle and stage reference
  • scrollbar doesn't create any mask for the holder clip and can be created dynamically or at flash authoring environment using mask layer
  • any display objects can be set as scrollbar controls - forward and backward buttons - see setControls, setControlButtons method, track button and track - see setControls method. All of them are optional and scrollbar can work without them if mouse wheel or auto content scrolling on roll over is enabled
  • if you need to scroll content when mouse rolls over control buttons without clicking on them you can use scrollOnButtonRollOver property. Just set it to true, cause it is false by default
  • if you have any pixel graphics or fonts in your content and need to set only integer values you can use roundedPositionValues property by setting it to true
  • if your content holder clip dimensions were changed you should update scrollbar to position it correctly. Use update() method to do that. You can configure scrollbar to automatically update its properties by setting autoUpdate property to true
  • when you need to remove your scrollbar you can use remove() method to remove all event listeners created by the scrollbar. If you use flash authoring tool and your movieclip is removed automatically in animation you can listen for the removedFromStage display object event and call remove() in the event handler
  • you can also extend advanced scrollbar class by overridding scrollContent() and scrollIsNeeded() protected methods as any public methods


Example
Create advanced scrollbar instance. Pass contentHolder - movieclip with content, mask rectangle and stage to the constructor.

Optionally in constructor you can set:

  • scrollbar direction (horizontal or vertical, it is vertica by default
  • enable wheel (true by default)
  • enable auto scrolling on content roll over (false by default)
  • enable scrollbar auto update (false by default)
  import com.moto.template.common.view.AdvancedScrollbar;
    var myScrollbar:AdvancedScrollbar = new AdvancedScrollbar(
  contentHolder, mask.getRect(this), stage);
    // Set scrollbar controls
  // You need to pass track button and guide to add basic controls for
  // your scroll. The only requirement - track button and guide anchor
  // points should be set to the top left corner and they should be
  // placed in the same clip. Optionally you can set track to add
  // track click logic, control buttons and control holder which will be
  // hidden when scrollbar is not needed.
  myScrollbar.setControls(trackGuide, trackButton, track, bottomArrow, topArrow, controlsHolder);
    // Set only control buttons and their holder - used when you don't
  // need track and button, only arrows to control scrollbar.
  myScrollbar.setControlButtons(bottomArrow, topArrow, controlsHolder);
    // Scrolling speed ratio
  myScrollbar.ratio = 0.4;
    // Allow scrolling using mouse press
  myScrollbar.scrollOnPress = true;
    // Delay before recalling event of pressed button when it's jammed
  myScrollbar.onPressScrollingDelay = 80;
    // Auto update scroll if content dimensions were changed
  myScrollbar.autoUpdate = true;
    // Allow scrolling using mouse wheel
  myScrollbar.mouseWheelEnabled = true;
    // Enable auto scrolling on control buttons roll over
  myScrollbar.scrollOnButtonRollOver = true;
    // Enable auto scrolling on content roll over
  myScrollbar.autoScrollOnRollOver = true;
    // Holds all the scrollbar controls and controls graphics
  myScrollbar.scrollbarHolder = controlsHolder;
    // Round content x or y position if needed (when using pixel fonts)
  myScrollbar.roundedPositionValues = true;
    // Roll over area size when using auto scrolling option
  myScrollbar.autoScrollingAreaPercent = 0.3;
    // Use tweens with easing when moving or scrolling content
  myScrollbar.tweenWithEasing = true;
    // Use tweens with easing when moving track button
  myScrollbar.tweenTrackButtonWithEasing = true;
    // Set scrolling step when mouse wheel is used, multiplied by delta
  myScrollbar.wheelScrollingStep = .03;
    // Forward and backward buttons scrolling speed
  myScrollbar.buttonsScrollingStep = 0.15;
    // Auto scrolling (on roll over) speed
  myScrollbar.autoScrollingStep = 0.15;
    // Resize track button depending on content size
  myScrollbar.resizeTrackButtonDependingOnContent = true;
    // Manually updates scrollbar
  myScrollbar.update();
    // Memory clean up by removing unnecessary event listeners
  myScrollbar.remove();
