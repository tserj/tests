The class which implements the general logic for animated itemRenderers.

The general logic of itemRenderer's work is implemented in this class. It can be extended to a more complicated animation by inheriting this class and:

  • Implement override methods over and out and describe the compliant logic of the image change.
  • Implement override method changeState(state:String) and describe the compliant logic of the image change depending on the dedicated state.
  • Describe the additional animation logic on the timeline; this method is especially suitable in the cases when there are embedded MovieClips with their timelines.
