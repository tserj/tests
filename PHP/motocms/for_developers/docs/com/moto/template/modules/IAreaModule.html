Interfacepublic interface IAreaModule extends IMotoModule, IMotoObject, ICPTransformable, ITransformable, IIsDisplayObject, IRasterizable, IPropertyUpdatable, IHasPathPrefix, flash.events.IEventDispatcher
SubinterfacesIRichContentModule
ImplementorsInfoModule

The IAreaModule interface is implemented by all Moto components which need to be scrolled.

This interface expresses content that is intended for scrolling.
