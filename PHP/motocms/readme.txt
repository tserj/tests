Moto CMS is a revolutionary content management system aimed at Flash developers 
and designed to simplify the publication of Web content to Flash websites. Moto CMS 
was built with no limits in animation and interactivity in mind. It gives Flash developers 
an opportunity to create Flash websites that can be easily edited by the end-users. 


///////////////////////////////////////////////////////////////////////////////////////////////
//  What is in the box:
///////////////////////////////////////////////////////////////////////////////////////////////

* website_example (Ready website with the control panel, just upload it to your server)

* for_developers (Files for flash developers)
    * components
        - MotoCMS.mxp (Flash IDE Components)
    * controlPanel
        - admin (Moto CMS Control Panel)
    * docs (Moto AS3 classes documentation)
    * examples (Website examples)
    * templates (Templates for building websites)

* readme.txt


///////////////////////////////////////////////////////////////////////////////////////////////
//  Requirements
///////////////////////////////////////////////////////////////////////////////////////////////

Moto CMS Control Panel Requirements:
    - Web server (can be local)
    - PHP version 5.2 or 5.3
    - Zend Optimizer 3.3 (for PHP 5.2 servers), or Zend Guard Loader (for PHP 5.3 servers) 
    - Flash Player 9 or higher

Flash CS3 or higher is required to install Moto CMS component and run the examples.
	
///////////////////////////////////////////////////////////////////////////////////////////////
//  Installation
///////////////////////////////////////////////////////////////////////////////////////////////

Quick Start:

Upload the contents of website_example folder to your server (ex. to the root folder of 
http://yourwebsite.com/) Now you can access your website at http://yourwebsite.com/ and your 
control panel at http://yourwebsite.com/admin. (By accessing your control panel for the first 
time you will be asked for its activation.) Please activate your product at 
http://accounts.cms-guide.com/ (Your account details will be sent to you after Moto CMS 
purchase) and then refresh browser page.

Read more about installation at http://support.cms-guide.com/entries/20412438-installation
Or see our video tutorials at http://support.cms-guide.com/forums/20441957-installation



For flash developers:

- use Adobe Extension Manager to install Moto CMS Flash components (MotoCMS.mxp)

- start from the website examples, try to compile them with Flash IDE. Use moto.flp - flash 
project to compile both .fla files simultaneously.

- copy Moto CMS Control Panel admin folder to the server. Copy any website example to the same 
folder overwriting the existing files and go to http://yourwebsite.com/admin/ to access your 
control panel (By accessing your control panel for the first time you will be asked for its 
activation.) Please activate your product at http://accounts.cms-guide.com/ (Your account details 
will be sent to you after Moto CMS purchase) and then refresh browser page.
