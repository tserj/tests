Moto CMS – революционная система управления контентом, предназначенная для Flash разработчиков и 
направленная на то, чтобы упростить размещение контента на флеш сайтах.  Moto CMS позволяет без 
ограничений использовать анимацию и интерактивные элементы. Система позволяет веб-разработчикам 
создавать Flash сайты, которые легко могут быть отредактированы конечными пользователями. 


///////////////////////////////////////////////////////////////////////////////////////////////
//  Что в коробке:
///////////////////////////////////////////////////////////////////////////////////////////////

* website_example (готовый веб-сайт с Панелью управления, просто загрузите его на свой сервер)

* for_developers (файлы для flash разработчиков)
    * components
        - MotoCMS.mxp (Flash IDE компоненты)
    * controlPanel
        - admin (Панель управления Moto CMS)
    * docs (Moto AS3 классы документации)
    * examples (примеры сайтов)
    * templates (шаблоны для создания сайтов)

* readme.txt
* readme_ru.txt


///////////////////////////////////////////////////////////////////////////////////////////////
//  Требования
///////////////////////////////////////////////////////////////////////////////////////////////

Требования Панели управления Moto CMS:

    - Веб-сервер (может быть локальным),
    - PHP 5.2 или 5.3
    - Zend Optimizer 3.3 (для серверов с РНР 5.2) или Zend Guard Loader (для серверов с РНР 5.3) 
    - Flash Player 9 или выше
	
///////////////////////////////////////////////////////////////////////////////////////////////
//  Установка
///////////////////////////////////////////////////////////////////////////////////////////////

Быстрый старт:

Загрузите содержимое папки website_example на сервер (например, в корневую папку http://yourwebsite.ru/) 
После этого перейдите к http://yourwebsite.ru/admin/, чтобы получить доступ к Панели управления. 
(Получая доступ к Панели управления в первый раз вам будет предложено его активировать). Пожалуйста, 
активируйте ваш продукт на http://accounts.cms-guide.com/ (Детали вашей учетной записи будут отправлены 
вам после покупки Moto CMS), а затем обновите страницу браузера.

Узнайте больше об установке на 
http://support.cms-guide.com/entries/20412438-installation
А также смотрите наши видео туториалы на http://support.cms-guide.com/forums/20441957-installation



Для flash разработчиков:

 - с помощью Adobe Extension Manager установите компоненты Moto Flash CMS  (MotoCMS.mxp).

- начните работу с примеров сайтов, попробуйте скомпилировать их с Flash IDE. Используйте moto.flp 
для одновременной компиляции .fla файлов. 

- из папки «controlPanel» скопируйте папку «admin» с Панелью управления Moto CMS на сервер. Затем 
скопируйте содержимое любого примера сайта (например: examples\websiteExample_04) в эту же папку, 
с перезаписью существующих файлов, и перейдите к http://yourwebsite.ru/admin/, чтобы получить доступ 
к Панели управления. (Получая доступ к Панели управления в первый раз вам будет предложено его активировать).
Пожалуйста, активируйте ваш продукт на http://accounts.cms-guide.com/ (Детали вашей учетной записи 
будут отправлены вам после покупки Moto CMS), а затем обновите страницу браузера.
